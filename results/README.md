# Results

This directory contains figures and logs generated by `code/distilbert_comparison.ipynb`.

## Figures (`figures/`)

| File | Description |
|---|---|
| `01_accuracy_comparison.png` | Our results vs. paper-reported accuracy for DistilBERT and BERT-base on IMDb |
| `02_parameter_count.png` | Parameter count (millions) with annotated % reduction |
| `03_model_size.png` | Model memory footprint in MB (FP32) |
| `04_inference_latency.png` | Per-sample inference latency on GPU (ms) |
| `05_training_time.png` | Fine-tuning wall-clock time (minutes) |
| `06_accuracy_vs_size.png` | Scatter: accuracy vs. parameter count, bubble size = latency |
| `07_radar_chart.png` | Multi-axis radar chart across all metrics |
| `08_summary_4panel.png` | 4-panel summary figure (accuracy, params, latency, training time) |

## Summary

| Metric | DistilBERT | BERT-base |
|---|---|---|
| Test Accuracy | 93.06% | 94.11% |
| Parameters | 67.0M | 109.5M |
| Model Size | 255 MB | 418 MB |
| GPU Inference Latency | 6.58 ms | 12.46 ms |
| Training Time | 19.5 min | 38.6 min |

DistilBERT retains **98.9%** of BERT accuracy with **38.8% fewer parameters** and **1.9× faster** GPU inference.
